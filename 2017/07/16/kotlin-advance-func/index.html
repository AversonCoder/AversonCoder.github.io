内联函数

每次使用 Lambda 的时候都会创建一个额外的类。这导致了效率的低下。使用内联函数可以避免该问题。内联函数适用于参数为 Lambda 且频繁调用的情况。

注意：简单函数 Kotlin 不建议使用内联
